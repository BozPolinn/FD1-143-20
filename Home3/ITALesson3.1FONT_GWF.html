Способы включения шрифтов в документ

Способ 1

Шрифт можно включить в документ при помощи ссылки на ресурс Google Fonts

    Для этого необходимо:
  • Зайти на ресурс Google Fonts;
  • Выбрать наиболее подходящий шрифт;
  • Выбрать несколько начертаний (но не слишком много, чтобы не превышать адекватное время загрузки);
  • Открыть вкладку Выбор данного стиля;
  • Скопировать ссылку на шрифт;
  • Включить в документ ссылку на шрифт в теге 'link' раздела 'head';

Способ 2

Разместить шрифт на сервере, откуда впоследствии браузер клиента его загрузит

    Для этого необходимо:
  • Зайти на ресурс Google Fonts;
  • Выбрать наиболее подходящий шрифт;
  • Открыть вкладку Выбор данного стиля;
  • Выбрать несколько начертаний (но не слишком много, чтобы не превышать адекватное время загрузки);
  • Загрузить документ со шрифтом;
  • Конвертировать документ в форматы ttf, eot, otf, woff, woff2, svg, чтобы браузер клиента мог прочитать один из них;
  • Добавить документ .html ссылки на все эти файлы.
